BibTeXとは

BibTeXは，論文・書籍・Webページなどの文献情報を決まった形式で管理するための記述方式です。 LaTeXや文献管理ツールと組み合わせることで，本文中の引用と参考文献リストを自動で生成できます。

何のため？ 文献情報をまとめて管理し，参考文献リストを自動生成するため。
何が入る？ 著者名，タイトル，発表年，会議名，雑誌名，ページ番号，DOIなど。
何が大事？ 取得元を選び，取得後に必ず中身を確認すること。

BibTeXの種類

種類 対象 使う場面
@article 雑誌論文 Journal論文を引用するとき
@inproceedings 会議論文 国際会議・国内会議の論文を引用するとき
@book 書籍 教科書や専門書を引用するとき
@misc その他 Webページ，arXiv，技術資料などを引用するとき

BibTeXの主要フィールド

フィールド 意味 注意点
author 著者名 英語では “姓, 名” または “名 姓” の形式に注意する
title 論文タイトル CLIP，BERT，ImageNetなどの大文字は波括弧で保護する
booktitle 会議名 @inproceedings で使う。会議名の正式表記を確認する
journal 雑誌名 @article で使う。雑誌名，巻，号，ページを確認する
year 発表年 arXiv公開年ではなく，採択・出版年を使う場合がある
pages ページ番号 範囲は 12--14 のようにハイフン2つで書くことが多い
doi DOI 可能なら入れる。公式ページで確認する
url URL WebページやarXivでは有用。論文なら公式URLが望ましい

BibTeXの入手方法

基本的には，各論文ページにある Cite / 引用 / Export Citation / BibTeX から取得できます。 ただし，取得元によって正確性が異なるため，優先順位を意識する必要があります。

入手元 信頼度 使い方 注意点
学会・出版社の公式ページ 最優先 Cite / BibTeX / Export Citation から取得 基本的に最も信頼できる
DBLP 便利 BibTeXリンクから取得 形式は整っているが，公式情報との照合は必要
arXiv 要確認 arXivのExport BibTeXから取得 採択版があるなら，学会・出版社版を優先する
Google Scholar 非推奨 引用 → BibTeX から取得 著者名，会議名，年，DOIなどが誤っていることがある
基本方針： まず公式ページから取得し，なければDBLPを確認する。Google Scholarは最後の手段として使い，必ず中身を確認する。

引用情報の書き方

BibTeXは文献情報のデータですが，最終的には本文中の引用と参考文献欄に反映されます。 重要なのは，読者が元の文献にたどり着けることです。

本文中の引用

書き方 例 使う場面
著者を主語にする Radfordらは，画像と言語を対応付けるCLIPを提案した。 その研究の主張や貢献を説明したいとき
文末で引用する 画像と言語の対応付けにはCLIPが広く用いられている。 背景説明に根拠を付けたいとき
複数文献をまとめる 近年，視覚言語モデルを用いた研究が進んでいる。 関連研究の流れを示したいとき

参考文献欄に必要な情報

文献タイプ 必要な情報
会議論文 著者，タイトル，会議名，ページ，年，DOI
雑誌論文 著者，タイトル，雑誌名，巻，号，ページ，年，DOI
書籍 著者，書名，出版社，年
Webページ 著者または組織名，タイトル，URL，閲覧日
arXiv 著者，タイトル，arXiv番号，年。ただし採択版があればそちらを優先

整形の注意点

タイトルの大文字を保護する

BibTeXでは，スタイルによってタイトル中の大文字が小文字に変換されることがあります。 CLIP，BERT，ImageNetなどは波括弧で囲みます。 

title = {{CLIP}: Learning Transferable Visual Models from Natural Language Supervision}
ページ範囲はハイフン2つで書く

LaTeXではページ範囲を 12--14 のように書くことが多いです。 

pages = {12--14}
arXiv版と採択版を混同しない

arXivはプレプリントです。会議や論文誌に採択済みの場合は，採択版の情報を使う方が安全です。

会議名・雑誌名を統一する

同じ会議でも表記が揺れると，参考文献リストの見た目が悪くなります。 研究室や論文内で表記を統一しましょう。

BibTeX例

会議論文

@inproceedings{radford2021clip,
  author    = {Radford, Alec and Kim, Jong Wook and others},
  title     = {{CLIP}: Learning Transferable Visual Models from Natural Language Supervision},
  booktitle = {Proceedings of the International Conference on Machine Learning},
  pages     = {8748--8763},
  year      = {2021}
}

雑誌論文

@article{example2026journal,
  author  = {Yamada, Taro and Suzuki, Hanako},
  title   = {A Survey on Uncertainty Estimation},
  journal = {Journal of Machine Learning Research},
  volume  = {27},
  number  = {1},
  pages   = {1--20},
  year    = {2026}
}

Webページ

@misc{example2026web,
  author       = {{Example Organization}},
  title        = {Example Web Page Title},
  howpublished = {\url{https://example.com}},
  year         = {2026},
  note         = {Accessed: 2026-05-21}
}

まとめ

BibTeXは，文献情報を自動で扱うための便利な形式です。 しかし，取得したBibTeXが常に正しいとは限りません。 公式ページを優先し，著者名，タイトル，会議名，年，ページ，DOIを確認してから使うことが重要です。
